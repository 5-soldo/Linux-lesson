1.Написать скрипт, который удаляет из текстового файла пустые строки
и заменяет маленькие символы на большие. Воспользуйтесь tr или SED.

SED
~/lesson6$ touch text.txt  # создаю файл
~/lesson6$ nano text.txt   # вношу текст
~/lesson6$ sed -i '/./!d' text.txt   # удаляю все пустые строки

TR
nano del_str.sh  # создаю скрипт
#!/bin/bash
tr -d [:blank:] < $1 | tr a-z A-Z > $2  # удаляю пробелы, по конвееру отправляю на следующий скрипт
                                        # преобразую все маленькие символы в боьшие.
chmod +x del_str.sh  # предоставляю права на выполнение
./del_str.sh text.txt text2.txt  # выполняем скрипт

Например, 001.txt, содержащий текст «Файл 001», 002.txt с текстом «Файл 002» и т. д.

#!/bin/bash
for year in {2010..2017}
do
  mkdir $year
  for month in {1..12}
  do
    mkdir $year/$month
    for file_number in {001..003}

3.* Использовать команду AWK на вывод длинного списка каталога, чтобы отобразить только права доступа 
к файлам. Затем отправить в конвейере этот вывод на sort и uniq, чтобы отфильтровать все повторяющиеся строки.

~/lesson6/2010$ ls -li  # использую директорию из задания 2
~/lesson6/2010$ ls -li | awk '{print $2}'  # проверяю тот ли столбец выводится, что мне нужен (с правами нужен)
~/lesson6/2010$ ls -li | awk '{print $2}' | sort | uniq  # фильтрую по повторам
48
drwxrwxr-x
