4.* Создать группу developer и нескольких пользователей, входящих в неё. 
Создать директорию для совместной работы. Сделать так, чтобы созданные одними пользователями
 файлы могли изменять другие пользователи этой группы

~$ sudo addgroup developer  # создаю группу
~$ sudo adduser userdev1  # добаавляю пользователя
~$ sudo usermod -aG developer userdev1  # добавляю пользователя в группу
~$ sudo useradd -m -s /bin/bash userdev2
~$ sudo usermod -aG developer userdev2
~$ mkdir Devtest
~$ sudo chgrp developer Devtest  # переименовываю группу
~$ chmod 2771 Devtest/  # присваиваю права
~$ sudo chmod g+s Devtest/   
130267 drwxrws--x 2 vadim developer 4096 фев  3 20:23  Devtest
